# Football Data

[![Live on GitHub Pages](https://img.shields.io/badge/Website-Live_on_GitHub_Pages-success?logo=github&labelColor=1e1e2e)](https://justkesha.github.io/football-data/)
[![Language: Russian](https://img.shields.io/badge/Language-RU-orange?labelColor=1e1e2e)](https://github.com/justkesha/football-data)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue?labelColor=1e1e2e)](LICENSE)

Веб-приложение для просмотра футбольной статистики с [football-data.org](https://www.football-data.org).  
Проект выполнен в рамках тестового задания на стажировку в компанию [SimbirSoft](https://www.simbirsoft.com).

<br>

Живая версия доступна на GitHub Pages: [justkesha.github.io/football-data](https://justkesha.github.io/football-data) <img width="10px" height="1px" alt="" src="https://i.imgur.com/qFmcbT0.png" /><img height="32px" alt="Tenor Gif" src="https://media.tenor.com/a2m-Y3dLmD0AAAAi/mona-github-loading-github.gif" />

> Чтобы ключи не попали в репозиторий - в демо-версии используются [тестовые данные](src/api/mock/).

### Требования

- **Node.js** 18+ _(проверено на v18.16.0)_
- **npm** 9+ _(проверено на 9.5.1)_

### Запуск

    npm start

> [!NOTE]  
> При трудностях с доступом к API с localhost воспользуйтесь веткой [cors-proxy](https://github.com/JustKesha/football-data/tree/cors-proxy).
